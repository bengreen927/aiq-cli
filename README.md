# AIQ CLI

Open source CLI tool that scans your personalized AI setup and produces a verifiable proficiency score.

## What It Does

AIQ evaluates the quality of your AI configuration — your custom skills, MCP servers, rules, automations, and memory systems — and produces a score that employers can verify.

## Installation

```bash
pip install aiq
```

## Usage

```bash
# Initialize your profile
aiq init

# Scan your local AI setup
aiq scan

# Run a full evaluation
aiq evaluate
```

## Privacy

AIQ is open source so you can audit exactly what data it collects. Nothing leaves your machine without your explicit approval. See [Privacy](https://aiq.dev/privacy) for details.
